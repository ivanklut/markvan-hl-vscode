1. Простой (Передача папки)
Если нужно поделиться с коллегой или другом прямо сейчас:
Заархивируй свою папку с расширением (где лежат package.json и папка syntaxes).
Твой знакомый должен распаковать её в папку расширений VS Code на своём компьютере.
Windows: %USERPROFILE%\.vscode\extensions
macOS/Linux: ~/.vscode/extensions После перезапуска VS Code у него всё заработает.



2. Удобный (Создание файла .vsix)Это создание «установочного файла» (как .exe или .apk), который можно просто перетащить в окно VS Code.
Установи инструмент упаковки: npm install -g @vscode/vsce.В терминале в папке своего проекта введи: vsce package.Появится файл markvan-0.0.1.vsix. Его можно пересылать в Telegram или по почте.Инсталляция: Меню Extensions (Ctrl+Shift+X) -> три точки в углу -> Install from VSIX...3. Официальный (Marketplace)Если ты хочешь, чтобы любой человек в мире мог найти твой язык через поиск в VS Code:Тебе нужно создать аккаунт в Azure DevOps и получить токен.Использовать команду vsce publish.Это чуть сложнее, требует иконки и описания в README.md, но зато это «взрослый» путь.

Шаг 2. Запустите режим отладкиНажмите клавишу F5 (или перейдите на левую панель во вкладку «Run and Debug» со значком жука и треугольника).Вверху этой панели нажмите на зеленую стрелочку с надписью «Launch Extension» (или Run Extension).Шаг 3. Что произойдет дальшеVS Code подумает пару секунд и откроет совершенно новое, второе окно редактора. В его заголовке будет написано [Extension Development Host].В этом новом окне откройте ваш тестовый файл с разметкой Маркван (с вашими заголовками ===, --- и т.д.).
